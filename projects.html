Animals - a command-line implementation of the game “20 questions”, similar to Akinator (but more limited). It was inspired by an old Linux/Unix game also called “Animals” that I can’t locate at the moment, and it uses a very similar interface. Unlike that program, which uses a tree structure for questions, my program stores an answer for each question for each animal, reducing the number of guesses required to figure out what animal you’re thinking of. It only comes with 3 animals, so you have to teach it more yourself. Download it here.
